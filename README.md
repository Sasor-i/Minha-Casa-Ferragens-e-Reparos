# Minha Casa Ferragens e Reparos

Este foi o primeiro site que desenvolvi do zero com uma proposta mais profissional, como parte de um trabalho acadêmico da faculdade.

O projeto foi criado com foco em apresentar a loja **Minha Casa Ferragens e Reparos** de forma clara, moderna e responsiva, facilitando o acesso às informações sobre o estabelecimento, seus produtos e formas de contato.

Durante o desenvolvimento, enfrentei desafios comuns para quem está iniciando na área de **Web Development**, o que tornou o processo uma ótima oportunidade de aprendizado. Ao longo do projeto, pude aplicar e aprimorar meus conhecimentos em **HTML, CSS e JavaScript**, além de trabalhar aspectos importantes como organização do código, responsividade, estrutura visual e experiência do usuário.

Fiquei satisfeito com o resultado inicial e pretendo continuar evoluindo o projeto futuramente, adicionando novas funcionalidades, melhorias no design, ajustes no código e possíveis integrações que tornem o site ainda mais completo e profissional.

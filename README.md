# Minha-Casa-Ferragens-e-Reparos
Este foi o primeiro site que desenvolvi do zero com uma proposta mais profissional, como parte de um trabalho acadêmico da faculdade.

Durante o processo, enfrentei alguns desafios comuns para quem está iniciando na área de Web Development, mas consegui aplicar meus conhecimentos em HTML, CSS e JavaScript para construir uma página funcional, responsiva e visualmente organizada.

Fiquei satisfeito com o resultado inicial e pretendo continuar aprimorando o projeto futuramente, adicionando novas funcionalidades, melhorias no design e ajustes no código conforme avanço nos estudos.
